**Date:** 2026-03-24
**Mode:** LITE (landing/brochure + multi-page portfolio)

## Goal
Rebuild utopiagroup.com.my as Malaysia's most impressive AI company website. Position as the only AI implementation partner that actually operates 30+ businesses.

## Customer Type
Malaysian SME owners and business decision-makers. B2B. English-language.

## Decisions Made

| Decision | Answer |
|----------|--------|
| Approach | Operator-First ("We Run AI Across 30 Companies") |
| Language | English |
| Structure | Multi-page: Home, /automate (5 dept sub-pages), /results, /about, /contact |
| Photos | Placeholder slots — CY provides later |
| Products | Organized by department (Accounting, HR, Marketing, Sales, Operations) |
| Pricing | Visible: Free → RM12K+RM3K/mo → RM68K All-In |
| Mac Mini | Hero differentiator — physical machine in client office |
| TrueAI | Branded feature (trademark), not standalone product |
| Case Studies | Real numbers + AI-specific results (creative freedom granted) |
| Design | Clean white, brand #116dff, ONE dark section (Mac Mini), mobile-first |
| CTA | WhatsApp primary (#25D366), "Try Free" secondary |
| Domain | utopiagroup.com.my |
| Stack | Next.js 16 + Tailwind 4 + TypeScript + Motion + Lucide |
